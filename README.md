# AWS-Cloud-CSN-Bootcamp-2025
My AWS Cloud CSN Journey

## Overview

The **AWS Cloud CSN Bootcamp (2025)** was an intensive, hands-on cloud computing program designed to build strong foundational and practical skills in **Amazon Web Services (AWS)**. The bootcamp focused on real-world cloud scenarios, core AWS services, security best practices, and cloud architecture fundamentals, preparing participants for entry-level cloud, DevOps, and cloud engineering roles.

This repository documents my **learning journey, labs, notes, and projects** completed during the bootcamp and serves as a portfolio reference for my growing cloud skill set.

---

## Program Objectives

* Build a strong foundation in **cloud computing concepts**
* Gain hands-on experience with **core AWS services**
* Understand **cloud security, identity, and access management**
* Learn how to design **scalable, reliable, and cost-effective architectures**
* Prepare for **AWS foundational-level certifications** and real-world cloud roles

---

## Key Skills & Technologies Learned

### Cloud Fundamentals

* Cloud computing models (IaaS, PaaS, SaaS)
* Benefits of cloud adoption
* AWS global infrastructure (Regions, AZs, Edge Locations)

### Core AWS Services

* **Compute:** EC2, AMIs, Auto Scaling basics
* **Storage:** S3, EBS, EFS (use cases and comparisons)
* **Networking:** VPC, Subnets, Route Tables, Internet Gateways, Security Groups
* **Databases:** RDS overview, database use cases

### Security & Identity

* AWS Shared Responsibility Model
* IAM Users, Groups, Roles, and Policies
* MFA and best security practices
* Securing EC2 and cloud resources

### Monitoring & Cost Management

* AWS CloudWatch basics
* AWS Pricing Calculator
* Cost optimization principles

### Operating in the AWS Cloud

* AWS Management Console
* AWS CLI (introductory usage)
* Resource provisioning and basic troubleshooting

## Hands-On Labs & Practical Work

Throughout the bootcamp, I completed multiple **guided and independent labs**, including:

* Launching and configuring **EC2 instances**
* Creating and securing **VPC networks**
* Attaching and managing **EBS volumes**
* Hosting static content using **Amazon S3**
* Implementing **IAM roles and permissions**
* Applying **security best practices** to cloud resources

Each lab emphasized real-world scenarios and reinforced both **technical skills** and **problem-solving abilities**.
